# サービスセクション用画像

このディレクトリには、サービスセクションの各カードで使用する背景画像を配置します。

## 必要な画像ファイル

### 背景画像（Background Images）
各サービスの背景として使用される画像です。**主役として表示されるため、視認性の高い画像を推奨します。**

1. **web-design-bg.jpg** - Webサイト制作
   - 推奨: Webサイトのモックアップ、デザインスケッチ、ブラウザウィンドウ
   - スタイル: アイソメトリック、低彩度、ミニマル
   - サイズ: 1200x800px以上推奨（高解像度）

2. **web-app-bg.jpg** - Webアプリ / システム開発
   - 推奨: ダッシュボード、コードエディタ、システムアーキテクチャ図
   - スタイル: テクニカル、クリーン、モノクロベース
   - サイズ: 1200x800px以上推奨（高解像度）

3. **automation-bg.jpg** - 業務改善・自動化
   - 推奨: ワークフロー図、自動化を表す抽象的な図、ツールアイコンの集合
   - スタイル: 抽象、幾何学的、グラデーション
   - サイズ: 1200x800px以上推奨（高解像度）

4. **learning-bg.jpg** - 学習支援・キャリア支援
   - 推奨: 成長カーブ、学習シーン、キャリアパス
   - スタイル: 温かみのあるイラスト、成長を表現
   - サイズ: 1200x800px以上推奨（高解像度）

## 画像の取得方法

以下の無料素材サイトから適切な画像を取得できます：

- **Unsplash** (https://unsplash.com/) - 高品質な写真
- **Pexels** (https://www.pexels.com/) - 無料の写真・動画
- **Freepik** (https://www.freepik.com/) - イラスト・アイコン
- **unDraw** (https://undraw.co/) - アイソメトリックイラスト
- **Heroicons** (https://heroicons.com/) - SVGアイコン

## デザインの特徴

- **ミニマルなデザイン**: アイコンを削除し、背景画像を主役に
- **視認性**: 背景画像は50%の透明度で表示（ホバー時60%）
- **アニメーション**: フェードインとスケールアニメーションで表示
- **グラデーション**: 背景画像の上にグラデーションオーバーレイを配置（ホバー時のみ表示）

## 注意事項

- 画像が存在しない場合、グラデーション背景のみが表示されます
- 背景画像は`background-size: cover`で全体をカバーします
- 画像は中央配置（`background-position: center`）されます
- 高解像度の画像を推奨します（Retinaディスプレイ対応）
